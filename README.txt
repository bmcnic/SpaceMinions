Steps for running Space Minions:

1. Download JDK at https://ninite.com/jdk/

2. Download JGRASP at http://spider.eng.auburn.edu/user-cgi/grasp/grasp.pl?;dl=download_jgrasp.html

3. Extract the downloaded .zip file

4. Open SpaceMinions.java 

5. Click the red guy at the top of the screen who looks like hes running to start the game.
